# Git Commands - Simple Analogy 🏠

## Moving into a New House Analogy

---

### 1. Navigate to your project folder
cd "C:\Users\xxx\OneDrive\Desktop\Lab Exp-1"
🚶 Walk to your current house
(Go to the folder where your files are)

---

### 2. Initialize Git
git init
📦 Get empty moving boxes
(Prepare the folder for Git tracking)

---

### 3. Add all files
git add .
📥 Pack ALL your belongings into boxes
(Add all files, ready to move)

---

### 4. Commit with message
git commit -m "Experiment 1 - Simple User Registration Form"
🏷️ Label the boxes with a description
(Save files locally with a message)

---

### 5. Rename branch to main
git branch -M main
🛣️ Name the main road to reach your new house
(Set the main branch name)

---

### 6. Add remote origin
git remote add origin https://github.com/YOUR_USERNAME/DevOps-Lab.git
🏠 Write down the new house address
(Tell Git where your GitHub repo is)

---

### 7. Push to GitHub
git push -u origin main
🚚 Send the moving truck to deliver all boxes
(Upload everything to GitHub)

---

## Summary
Pack → Label → Get Address → Deliver!
