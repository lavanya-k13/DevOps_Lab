# Experiment No. 3 - Practice Source Code Management on GitHub

## Aim
Practice source code management on GitHub using the 
source code written in Exercise 1.

## Software Required

| Software / Tool | Purpose |
|----------------|---------|
| Git | Local version control system |
| GitHub Account | Remote repository hosting |
| VS Code | Create and manage source code files |
| Web Browser | Access GitHub and repositories |
| Command Prompt | To execute Git commands |

## Installation Steps
1. Download Git from https://git-scm.com/downloads
2. Configure Git:
   git config --global user.name "Your Name"
   git config --global user.email "your.email@example.com"
3. Create GitHub account at https://github.com
4. Install VS Code from https://code.visualstudio.com

## Git Commands Used

### Clone Repository
git clone https://github.com/yourusername/user-registration-app.git

### Navigate to Project
cd user-registration-app

### Stage Changes
git add .

### Commit Changes
git commit -m "Added source code for simple user registration form"

### Push to GitHub
git push origin master
